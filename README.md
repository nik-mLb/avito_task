# avito_task

## Запуск 

**команда:** make start
запускает бд postgres на 5433 порту и сам сервер на 8080

## Тесты

**команда:** make test
выполняет тесты и создает папку coverage с cover.out

**команда:** make coverage 
созадет html и показывает покрытие (у меня 78%)
